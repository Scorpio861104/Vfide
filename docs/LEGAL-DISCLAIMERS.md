## Usage Guidelines

### Where to Place Disclaimers:

**MANDATORY Locations:**
1. ✅ Purchase/checkout page (full disclaimer + checkboxes)
2. ✅ Homepage hero section (short disclaimer)
3. ✅ Website footer (every page)
4. ✅ Terms of Service (comprehensive language)
5. ✅ Whitepaper/documentation (full disclaimer)

**RECOMMENDED Locations:**
1. ✅ About page
2. ✅ FAQ page
3. ✅ Social media profiles/bios
4. ✅ Email signatures
5. ✅ Marketing materials
6. ✅ Announcements and blog posts

**OPTIONAL but Good Practice:**
1. ✅ GitHub README
2. ✅ Discord/Telegram channel descriptions
3. ✅ Medium articles
4. ✅ YouTube video descriptions
5. ✅ Press releases

### Disclaimer Hierarchy:

**Level 1 - Critical (Must Have):**
- "NOT investment securities"
- "May lose all value"
- "Utility tokens for protocol use only"

**Level 2 - Important (Should Have):**
- "No profit guarantees"
- "Active participation required"
- "Consult advisors"

**Level 3 - Additional (Nice to Have):**
- Specific risk disclosures
- Jurisdiction restrictions
- Forward-looking statement warnings
