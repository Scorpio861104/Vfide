# Security Policy
# 
# If you discover a security vulnerability, please report it to security@vfide.io
# 
# Please include:
# - Vulnerability type
# - Steps to reproduce
# - Potential impact
# - Your contact information
#
# We will respond within 48 hours and work with you to address the issue.
#
# Please do not disclose the vulnerability publicly until we have had a chance
# to address it.
#
# We appreciate your help in keeping VFIDE secure!
