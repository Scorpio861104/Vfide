# Future-Tier Contracts

Contracts in this directory are intentionally excluded from the active production deployment scope.

## Policy

- Do not include these contracts in production deployment inventories by default.
- Promotion from future-tier to production requires:
  1. Explicit security review
  2. Deployment-script wiring
  3. Production inventory updates
  4. Frontend ABI parity review
