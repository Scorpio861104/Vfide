# VFIDE Badge Artwork Specifications

> Design System for 27 Achievement Badges
> Each badge is a soulbound NFT that represents EARNED reputation

---

## 🎨 Design Philosophy

**Core Principle:** "Trust is beautiful, scammers are ugly"

- **Earned badges** = Rich, detailed, prestigious designs
- **Rare badges** = Animated, glowing, legendary aesthetics
- **Renewable badges** = Subtle timer/refresh indicators

---

## 🏛️ Visual Identity

### Color Palette

| Category | Primary | Secondary | Accent |
|----------|---------|-----------|--------|
| Pioneer & Foundation | Gold #FFD700 | Deep Purple #4A0080 | White #FFFFFF |
| Activity & Participation | Electric Blue #00D4FF | Navy #001F3F | Cyan #00FFFF |
| Trust & Community | Emerald Green #50C878 | Forest #228B22 | Lime #32CD32 |
| Commerce & Merchants | Royal Purple #7851A9 | Violet #8B00FF | Silver #C0C0C0 |
| Security & Integrity | Crimson Red #DC143C | Obsidian #0B0B0B | Gold #FFD700 |
| Achievements & Milestones | Diamond White #E6E6FA | Platinum #E5E4E2 | Rainbow Gradient |
| Education & Contribution | Orange Amber #FFBF00 | Copper #B87333 | Warm White #FFF8DC |

### Shape Language

- **Pioneer badges**: Shield/crest shapes (historical, foundational)
- **Activity badges**: Circular with motion lines (energy, momentum)
- **Trust badges**: Hexagons (network, community interconnection)
- **Merchant badges**: Diamond shapes (value, commerce)
- **Security badges**: Octagon/stop-sign shapes (protection, defense)
- **Achievement badges**: Star bursts (excellence, milestone)
- **Contribution badges**: Gear/lightbulb shapes (creation, innovation)

## 📐 Technical Specifications

### File Formats
- **SVG**: Vector source files (scalable)
- **PNG**: 512x512, 256x256, 128x128, 64x64
- **WebP**: Optimized for web
- **GIF/MP4**: For animated versions

### Metadata Schema (ERC-721)
```json
{
  "name": "VFIDE Pioneer #2847",
  "description": "One of the first 10,000 VFIDE community members. Earned through early adoption, commemorated forever.",
  "image": "ipfs://Qm.../pioneer/2847.png",
  "animation_url": "ipfs://Qm.../pioneer/2847.mp4",
  "external_url": "https://vfide.io/badge/pioneer/2847",
  "attributes": [
    {"trait_type": "Category", "value": "Pioneer & Foundation"},
    {"trait_type": "Rarity", "value": "Legendary"},
    {"trait_type": "Badge Number", "value": 2847},
    {"trait_type": "Mint Date", "display_type": "date", "value": 1734441600},
    {"trait_type": "ProofScore Boost", "value": 30},
    {"trait_type": "Permanent", "value": true}
  ]
}
```

### Display Contexts
1. **Profile Avatar Frame**: Badge as profile border
2. **Card View**: Full badge with name/description
3. **Inline Icon**: Tiny version next to username
4. **NFT Gallery**: Full animated display
5. **Social Share Card**: Optimized for Twitter/Discord
